pongalytics
===========

Statistics for pong

# Install

```bash
git clone https://github.com/jessegavin/pongalytics.git && cd $_
npm install
```

# Usage: development

1. Install heroku's dev toolbelt: [foreman](https://toolbelt.heroku.com/)
1. Start server: `foreman start`
1. Visit [http://localhost:3000/](http://localhost:3000/)
